# 안녕하세요, 김관호입니다

AI 기반 실시간 협업 시스템 개발에 관심을 가진 백엔드 개발자입니다.  
보이지 않는 서버 영역에서 클라이언트와의 통신을 설계하고 처리하는 과정에서 흥미를 느끼며 백엔드 개발에 집중하고 있습니다.  

특히 WebSocket을 활용한 실시간 데이터 처리와, AI를 활용한 기능 자동화에 관심이 있습니다.  
현재는 AI 기반 칸반 보드 시스템을 개발하며, 사용자 생산성을 높이는 서비스를 만드는 것을 목표로 하고 있습니다.  

AI 기술의 발전에 맞춰, 이를 실제 서비스에 적용하여 더 편리하고 효율적인 시스템을 만드는 개발자로 성장하고자 합니다.

## 프로젝트

### AI 기반 실시간 협업 칸반 시스템
- LLM(OpenAI API)을 활용한 Task Decomposition 기능 구현
- WebSocket(Socket.io)을 이용한 실시간 상태 동기화
- 칸반 보드 기반 프로젝트/태스크 CRUD 기능 구현

**기술적 특징**
- WebSocket을 활용하여 사용자 간 작업 상태를 실시간으로 동기화
- AI를 활용하여 작업을 자동으로 분해하고 효율적인 업무 흐름 제공

## 기술 스택

### Frontend
- React: 컴포넌트 기반 UI 설계 경험
- Next.js: 프로젝트 구조 설계 및 페이지 구성 경험
- Tailwind CSS: 유틸리티 기반 스타일링 적용

### Backend
- Node.js / Express: REST API 설계 및 서버 구축 경험

### Realtime
- WebSocket (Socket.io): 실시간 채팅 및 상태 동기화 구현

### Database
- MongoDB: 유연한 스키마 기반 데이터 모델링 경험

### AI
- OpenAI API: 작업 분해(Task Decomposition) 기능 구현
