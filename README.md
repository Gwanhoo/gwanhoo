# 안녕하세요, 김관호입니다

AI 기반 실시간 협업 시스템 개발에 관심을 가진 백엔드 개발자입니다.  
보이지 않는 서버 영역에서 클라이언트와의 통신을 설계하고 처리하는 과정에서 흥미를 느끼며 백엔드 개발에 집중하고 있습니다.  

특히 WebSocket을 활용한 실시간 데이터 처리와, AI를 활용한 기능 자동화에 관심이 있습니다.  
현재는 AI 기반 칸반 보드 시스템을 개발하며, 사용자 생산성을 높이는 서비스를 만드는 것을 목표로 하고 있습니다.  

AI 기술의 발전에 맞춰, 이를 실제 서비스에 적용하여 더 편리하고 효율적인 시스템을 만드는 개발자로 성장하고자 합니다.

## 기술 스택

### Frontend
- React: 컴포넌트 기반 UI 설계 경험
- Next.js: 프로젝트 구조 설계 및 페이지 구성 경험
- Tailwind CSS: 유틸리티 기반 스타일링 적용

### Backend
- Node.js / Express: REST API 설계 및 서버 구축 경험

인증, 게시글, 채팅, 스터디룸 등 기능별로 RESTful API를 설계하여 구현했습니다.  
총 40개 이상의 엔드포인트를 기능 단위로 분리하여 관리하고 있습니다.

### Database
- MongoDB: 유연한 스키마 기반 데이터 모델링 및 컬렉션 설계

커뮤니티 및 실시간 채팅 기능 구현을 위해 MongoDB를 사용하여 데이터 구조를 설계했습니다.

- chat / messages: 실시간 채팅 메시지 저장
- chatroom: 채팅방 단위 관리
- post / comment: 게시글 및 댓글 기능 분리
- user / people: 사용자 및 참여자 정보 관리
- invitation: 협업 초대 기능 지원
- sessions: 사용자 세션 관리

각 기능을 독립적인 컬렉션으로 분리하여 확장성과 유지보수성을 고려했으며,  
인덱스를 적용하여 조회 성능을 최적화했습니다.
